
通信算法实现可以分解为很多的标准算法模块，例如匹配滤波、频偏校正、Viterbi译码等，站在芯片设计者的角度，任何复杂的逻辑只是一堆基本的逻辑运算单元组合.

数字逻辑设计主要涵盖几大内容;

1. 滤波器类：  
（1）FIR滤波器；   
（2）IIR滤波器；   
（3）HB滤波器；   
（4）CIC滤波器；   
2. 逼近函数类：  
（1）CORDIC算法；  
（2）sin/cos；  
（3）Exp/Log；  
（4）Polar: atan / 平方根；  
（5）Rotator：向量旋转计算，如频偏补偿等；  
3. 数据变化类：  
（1）FFT： OFDM系列中的最核心的模块；  
（2）DCT：图像处理中的核心模块；  
（3）FWT: 快速WALSH变换，802.11b中CCK解码的关键；  
4. 纠错编码类：  
（1）CRC；  
（2）卷积码；  
（3）交织与打孔；  
（4）PN序列： 伪随机发生器；  
5. 存储类：  
（1）同步FIFO；  
（2）异步FIFO；  
6. 组合模块：  
（1）DDS：直接数字频率合成技术；  
（2）DDC/DUC： 数字上下变频；
